Funções das TAGS

● TABLE = é usada para definir tabelas. As tabelas são constituídas por linhas e por células. A etiqueta que significa table row é usada para definir uma linha. A etiqueta que significa table data é usada para definir uma célula.
● TH = O elemento HTML define uma célula cabeçalho do grupo de células de sua tabela
● TR =TR define uma linha numa tabela
● TD =TD significa "Table Data" e define cada uma das células da
Estas tags devem sempre surgir dentro de tags TR e, no seu interior ● THEAD = é utilizado para destacar os títulos das colunas de uma tabela.
● TBODY = é para ser utilizada no lugar de quando o conteúdo da célula for um cabeçalho, ao invés de dados.
● TFOOT = Elemento para o rodapé da tabela.
● ALIGN = sendo usada para mudar o alinhamento do texto.
● BORDER = essa propriedade é responsável por adicionar bordas em qualquer elemento HTML, seja ele uma div, uma imagem, um box, ou seja, qualquer elemento.
● CAPTION = representa o título de uma tabela
● ROWSPAN = servem para expandir as células fazendo com que uma única célula ocupe mais de uma coluna.
● COLSPAN = servem para expandir as células fazendo com que uma única célula ocupe mais de uma linha ou mais de uma linha.
